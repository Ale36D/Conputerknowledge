1.2计算机的基本组成

1.2.1 冯诺伊曼计算机的特点⭐️

  • 1945年冯诺伊曼在研究EDVAC机提出了"存储程序"的概念,以此概念为基础的各类计算机通称为冯诺伊曼机

💥重点

  • 计算机由运算器,存储器,控制器,输入输出设备五大部件注成

  • 指令和数据以同等地位存放在存储器中并可以按照地址访问

  • 指令和数据均用二进制数表示

  • 指令由操作码和地址码组成,操作码用来表示操作的性质,地址码用来表示操作数在存储器中的位置

1.2.2计算机的硬件框图
💥重点

  • 经典冯诺依曼计算机结构💥

  • 现代计算机以存储器为中心💥

😯知识点

  • 运算器和控制器一般集成在同一芯片上,因此统称为中央处理器(Central Processing Unit, CPU)

  • 把输入设备和输出设备简称为I/O设备

  • 现代计算机可认为由三大部分组成:CPU,I/O设备及主储存器(Main Memory, MM)

  • CPU和主存储器合起来又可称为主机,I/O设备又可称为外部设备

 

  • ⭐️ 现代计算机的组成框图

 

  • 算数逻辑单元(ALU)简称算逻部件,用来完成算数逻辑运算.

  • 控制单元(CU)用来解释存储器中的指令,并发出各种操作命令来执行指令.

  • ALU和CU是CPU的核心部件

  • I/O设备也受CU控制,用来完成相应的输入,输出操作

由此总结出,计算机的自动工作是在控制器的统一指挥下完成的

1.2.3计算机的工作步骤

  1. 建立数学模型

  2. 确定计算方法

  3. 编制解题程序

 

💥重点

  • 细化的计算机组成框图

  • 主储存器😋

主存储器(简称内存或主存)包括储存体M,各种逻辑部件及控制电路等

存储步长可以是8位,16位或32位等

一个存储字还可代表一条指令

主存的工作方式就是按存储单元的地址号来实现对存储字各位的存(写入),取(读出)

这种存储方式称为按地址存取方式,即按地址访问存储器(简称访存)

当需要反复使用某个数据或指令,只需要指出齐对应单元地址号即可,不必要占用更多的存储单元重复存放同一个数据或同一个指令,大大提高了存储单元的利用率

 

⭐️由于指令和数据都是由存储单元地址号来反应,因此取一条指令和取一个数据的操作完全可以认为是相同的,这样就可以使用一套控制线路来完成两种不同的操作

 

⭐️主存中必须配置两个寄存器MAR和MDR

MAR(Memory Address Register)是存储器地址寄存器其位数对应存储单元的个数(如MAR为10位,则有2的10次方=1024个存储单元,记为1K)

MDR(Memory Data Register)是存储器数据寄存器用来存放从存储体某个单元取出的代码或准备往某个存储单元存入的代码,其位数与存储字长相等

 

😋随着硬件技术的进步,主存都制成大规模集成电路的芯片,而将MAR和MDR集成在CPU​

😋早期计算机存储字长一般和机器的指令字长和数据字长相等,所以访问一次主存便可取一条指令或一个数据

😋现在计算机指令字长是可变的数据字长也要求可变,所以长度不由存储字长来确定,而由字节的个数来表示(1个字节被定义为由8位二进制代码组成)此时存储字长,指令字长,数据字长三者可各不相同😂,但必须是字节的整数倍💓

 

  • 运算器😄

运算器至少包括三个寄存器和一个算术逻辑单元(ALU)

算术逻辑单元(arithmetic and logic unit) 是能实现多组算术运算和逻辑运算的组合逻辑电路，简称ALU

算术逻辑单元（Arithmetic&logical Unit）是中央处理器(CPU)的执行单元，是所有中央处理器的核心组成部分，由"And Gate"（与门） 和"Or Gate"（或门）构成的算术逻辑单元，主要功能是进行二位元的算术运算，如加减乘(不包括整数除法)。基本上，在所有现代CPU体系结构中，二进制都以补码的形式来表示。

百度百科

 

  • 控制器😉

控制器是计算机的中枢神经,由他指挥各部件自动,协调的完成工作

  1. 命令存储器读出一条指令称为取指过程(也称取指阶段)

  2. 对指令进行分析,称为分析过程(也称为分析阶段)

  3. 根据操作数所在的地址以及指令的操作码完成某种操作称为执行过程(也称执行阶段)

⭐️通常就是完成一条指令操作的取指,分析,执行三个阶段

 

⭐️控制器组成

  • I/O

I/O子系统包括各种I/O设备及其相应的接口,它接收CU发出的各种控制命令,并完成相应的操作

1.3计算机硬件的主要技术指标⭐️

  • 机器字长

是指CPU一次能处理的数据位数,通常与CPU的寄存器位数有关系

  • 存储容量

包括主存容量和辅存容量

主存容量是指主存中存放的二进制代码的总位数

​⭐️存储容量=存储单元个数✖存储字长

辅助容量通常用字节来表示

举个栗子,1G = 1024M = 2^10×2^20= 2^30

  • 运算速度

计算机的运行速度与许多因素有关系

  1. 机器的主频

  2. 执行什么样的操作

  3. 主存本身的速度(主存速度快,取指,取数就快)

  4. .........

MIPS，即每秒执行多少百万条指令。MIPS = 指令条数/(执行时间×10)=主频/CPI。 MFLOPS，即每秒执行多少百万次浮点运算。MFLOPS=浮点操作次数/(执行时间×10)。 GFLOPS，即每秒执行多少十亿次浮点运算。GFLOPS=浮点操作次数/(执行时间×10)。 TFLOPS，即每秒执行多少万亿次浮点运算。TFLOPS=浮点操作次数/(执行时间×10)。
